Nonogram Website

nonogram is a solver of nonogram puzzles (also known as “griddler puzzles” or “paint by numbers”) for POSIX-compliant operating systems (e.g. Linux).
  • source package — nonogram_0.8.3.3.tar.gz (31 KiB)
  • Debian “etch” binary (i386) package — nonogram_0.8.3.3_i386.deb (13 KiB)
  • Debian “etch” data files package — nonogram-data_0.8.3.3_all.deb (23 KiB)
Usage example: 
$ nonogram <<EOF
6 3
3 2
1 2 1
2 3
3
1 1
2
2
1 1
3
EOF
